Frequently Asked Questions

What is FreeViewKodi?

FreeViewKodi is an unofficial Kodi add-on that allows you to manage and stream your own video libraries using simple JSON data files. It's designed to be a highly customizable and local-first solution for your media.

How do I add new content?

To add new content, you must edit the JSON files (e.g., `tvshows.json`, `movies.json`) located in your add-on's data folder. Each video item is an object in a JSON array. See the **Data Structure** section on the homepage for an example.

Is this add-on officially supported by Kodi or Freeview?

No, this add-on is an independent, unofficial project. It is not affiliated with, endorsed by, or supported by either the Kodi Foundation or Freeview.

What platforms does it work on?

The add-on should work on any platform that supports modern versions of Kodi, including Windows, macOS, Linux, Android, and Raspberry Pi.
