Installation Guide

Follow these simple steps to install the FreeViewKodi addon in your Kodi client. We cover both repository and manual installation methods.

Method 1: Install from Repository URL (Recommended)

This method allows you to receive automatic updates for the addon.

  1. From the Kodi home screen, navigate to **Settings** (the gear icon).
    Placeholder for Screenshot: Kodi Settings Screen
  2. In the Settings menu, go to **File manager**, then select **Add source**.
    Placeholder for Screenshot: Add Source Screen
  3. Click on the `` field and enter the repository URL: https://freeviewkodi.github.io/repo/.
  4. In the box below, name the source FreeViewKodi and click OK.
  5. Go back to the main **Settings** menu and select **Add-ons**.
  6. Click the **Package installer** icon (the open box icon), then choose **Install from zip file**.

    Important: You may need to enable "Unknown sources" in your Kodi settings before proceeding.

  7. Select the source you just added: FreeViewKodi.
  8. Select the repository file (e.g., repository.freeviewkodi-x.x.zip) and wait for the "Add-on enabled" notification.
  9. Go back one screen and select **Install from repository**. Choose the **FreeViewKodi Repository**.
  10. Navigate to **Video add-ons**, select **FreeViewKodi**, and click Install.

Method 2: Manual Installation from Zip

Use this method if you prefer to download the file directly.

  1. Download the latest version of the repository zip file from the project homepage or directly from this link: Download Zip File.
  2. Save the `.zip` file to a location you can access from your Kodi device.
  3. In Kodi, navigate to **Settings** → **Add-ons**.
  4. Click the **Package installer** icon and select **Install from zip file**.
  5. Browse to the location where you saved the downloaded zip file and select it.
  6. Wait for the "Add-on enabled" notification to confirm a successful installation.

Troubleshooting Common Issues

I get an "Unknown source" error when trying to install.

You need to enable "Unknown sources" in your Kodi settings. Go to **Settings** → **System** → **Add-ons** and toggle on the "Unknown sources" option.

The repository link is not working.

Double-check that you have typed the URL exactly as shown: https://freeviewkodi.github.io/repo/. Ensure there are no typos or extra spaces.

I am getting a "Dependencies not met" error.

This error typically means a required component of the addon is missing. This can sometimes be resolved by making sure your Kodi is up to date. You can also try a fresh installation of Kodi if the issue persists.
